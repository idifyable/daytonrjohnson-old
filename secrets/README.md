Database configuration variables kept in this folder.

The folder structure should be built out as follows:

```
.   secrets
|   +-- development
|   |   +-- POSTGRES_DB
|   |   +-- POSTGRES_USER
|   |   +-- POSTGRES_PASSWORD
|   +-- production
|   |   +-- POSTGRES_DB
|   |   +-- POSTGRES_USER
|   |   +-- POSTGRES_PASSWORD
```
